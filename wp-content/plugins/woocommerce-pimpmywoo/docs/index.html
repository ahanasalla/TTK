Description

PimpMyWoo helps you customise the way your WooCommerce shop looks without writing a single line of code or modify any template! It works with any standalone Wordpress installation, with any WordPress theme and of course with WooCommerce. PimpMyWoo is designed to facilitate online shop owners (from newbies to highly experienced ones) the WooCommerce customisation process.

You will be able to tweak the following:

  • any products listing page (from buttons to images, from fonts to borders and letter spacing)
  • any default products listing widget (from prices to images, from colours to font styles and text transforms)
  • the product page (from image galleries to badges, from background colours to font colours and text border styles)

With this plugin you can easily customise your WooCommerce look and feel to better fit your needs. Do you want to just have for Xmas some red Add to Cart buttons? It will take seconds to have it. Do you want to change the whole look of your store? It will take minutes. Immediately after the plugin activation you'll be able to access the customisation page. PimpMyWoo - WooCommerce Styler Plugin is well maintained and supported.

Installation and Usage

The installation steps are very easy and straight forward.

  1. Upload 'woocommerce-pimpmywoo' folder to the '/wp-content/plugins/' directory or upload the zip file via the 'Plugins -> Add New' menu
  2. Activate the plugin from the 'Plugin' page in Wordpress
  3. Activate your licese from 'WooCommerce -> PimpMyWoo' menu
  4. Enjoy customising your WooCommerce shop by accessing 'Appearance -> Customize' menu

After you navigate to the Customize page you will have PimpMyWoo menu and you can start browsing each section that you want to customise: default product listing, widget product listing or product page. The interface is using Wordpress builtin controls so it is very intuitive.
